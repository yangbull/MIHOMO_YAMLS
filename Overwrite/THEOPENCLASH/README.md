## 🚀 使用方法
1. 进入上方分类目录找到需要的 `.conf` 文件。
2. 复制文件 URL (Raw Link)。
3. 在 OpenClash -> 配置文件管理 -> 下载配置文件中粘贴 URL。
4. **重要**：在 OpenClash 插件设置 -> 覆写设置 -> 开发者选项 (或根据提示) 中设置对应订阅链接的环境变量 (`EN_KEY1`, `EN_KEY2` 等)。
